# 🇮🇳 Quiz Game on India

A **terminal-based quiz game** built in C++ that tests players' knowledge of Indian General Knowledge — covering history, geography, culture, politics, and more.

## ✨ Features

- 🎯 **Multi-category questions** — History, Geography, Politics, Culture, Sports
- 📊 **Live score tracking** — Points awarded for each correct answer
- 🏆 **Final score display** — Shows total score at the end of the quiz
- 💻 **Terminal interface** — Runs directly from the command line

![C++](https://img.shields.io/badge/C++-00599C?style=for-the-badge&logo=cplusplus&logoColor=white)

## 🚀 Getting Started

### Prerequisites

- A C++ compiler (g++, MinGW, or MSVC)

### Build & Run

```bash
# Clone the repository
git clone https://github.com/PaarthSuwi/Quiz-Game-on-India.git
cd Quiz-Game-on-India

# Compile the program
g++ -o quiz "Indian Quiz Game.cpp"

# Run the game
./quiz
```

> **Windows users:** You can also run the pre-compiled `Indian Quiz Game.exe` directly.

## 🎮 How to Play

1. Launch the quiz game
2. Read each question carefully
3. Enter the number corresponding to your answer
4. Get immediate feedback
5. View your final score at the end
